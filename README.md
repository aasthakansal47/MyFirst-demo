# 🛒 Amazon Clone (HTML & CSS)

## 🔍 Overview
This project is an Amazon-inspired e-commerce website built using only HTML and CSS. It replicates the user interface of Amazon and showcases Alexa-enabled devices with a clean and modern design.

---

## 🎯 Features
- Amazon-like homepage UI  
- Product sections for Alexa devices  
- Navigation bar with search functionality (UI)  
- Footer similar to Amazon  
- Responsive layout  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3
